# Smart Sorting: AI-Powered Produce Quality Detection 🍎

## 🎯 Overview
Smart Sorting is a deep learning-powered tool that classifies fruits and vegetables as **healthy or rotten** using a Flask web interface. It employs transfer learning with VGG16 for high accuracy and real-time performance.

---

## ✅ Features
- ✅ **High accuracy**: ~99% on test data  
- 🚀 **Support for 28 produce categories** (fruit + veggie pairs)  
- 🌐 **Web interface**: Upload image, view results instantly  
- 💾 **API endpoint**: Use REST (`/predict`, `/api/predict`, `/health`)  
- ⚙️ **Model format**: `.h5`, compatible with Keras/TensorFlow
